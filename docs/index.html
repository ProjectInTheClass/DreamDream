당신은 매일 아침 개운하게 일어나시나요?
당신의 숙면을 도와드릴께요!!



대부분의 사람들은 양질의 숙면이 인생에서 얼마나 큰 힘이 되는지 잘 알고 있습니다. 하지만 '꿀잠'은 결코 쉽지 않죠.

저희 DreamDream은 실제로 수면에 트러블을 겪고있는 개발자의 경험을 바탕으로 사람들이 어떻게 하면 '꿀잠'을 잘 수 있을까 하는 고민에서 시작하였습니다.

화면구성

Home

수면과 관련한 간단한 시간정보만을 제공하는 홈화면은 수면 방해요소를 최대한 제외하였습니다. 여러분은 버튼 하나만 누르고 그대로 잠에 드시면 됩니다.

Contents Library

Youtube와 연동하여, 수면에 도움을 줄 수 있는 수면 컨텐츠들을 저장 하여 언제든지 재생할 수 있습니다.

Sleep Analisys

Healthkit 으로부터 최근의 수면데이터를 불러와서, 당신의 수면습관을 한눈에 보여드립니다. 직접 개발한 수면 분석 알고리즘을 통해 적절한 수면관리 피드백을 제공합니다.

User Setting(개발중)

알람설정과 함께, 수면과 관련한 데이터를 입력하시면 더 체계적인 수면분석을 도와드립니다.

수면분석 알고리즘

불러온 수면데이터를 바탕으로, 사용자의 수면패턴을 분석함과 동시에 문제점을 찾아냅니다.
